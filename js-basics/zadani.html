2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  • JavaScript is Case Sensitive and Camel Case.
  • JavaScript uses variables (var, let, const).
  • JavaScript is dynamically typed like most other scripting languages.
  • For Single line comments we use "//" and for Multiple line comments we use at begining "/*" and for end we use "*/".
  • There are two types of variables global variable and variable in function.

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Vyskočí okno(popup) se zprávou */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Zadejte heslo: ');

confirm()

/* Potvrzení jestli chce uživatel přejít na stránky Seznam.cz */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování: Vyskakovací okna jsou blokovaná prohlížečem, protože můžou být otravována.
